# Trendyol Angular Frontend

A fully responsive Angular 20 frontend inspired by Trendyol, featuring product listings, sponsored sections, horizontal scrollers, and navigation.


## Features

- Home page with sponsored sections and horizontal scrollers.
- Designer clothing and shoes listings.
- Navigation bar with router links.
- Contact us page with information sections.
- Fully responsive design using Bootstrap 5.
- Angular 20 standalone components architecture.

## Prerequisites

- Node.js >= 18
- Angular CLI >= 20

## Build

To build the project for production:

```bash
ng build
```


## File Structure

- `src/app/` – Angular components and services.
- `src/assets/` – Images and static assets.
- `public/` – Additional assets for build.
